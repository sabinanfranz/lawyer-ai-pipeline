이전 출력이 스키마를 위반했다. 아래 조건을 만족하는 “유효 JSON 1개”만 즉시 반환하라.

- 출력은 반드시 JSON ONLY. 코드펜스/설명문/서론/꼬리말 금지.
- 허용 키는 정확히 2개: "title_candidates", "body_md_lines" (추가 키 금지)
- title_candidates: 3~6개. 비어있으면 즉시 3개를 생성하라(중복 금지).
- body_md_lines: 최소 10줄. 각 요소는 한 줄이며 개행 문자 포함 금지. 빈 줄은 ""로 표현 가능.
- 글은 Hook 2~3줄 + Body(3~6포인트) + CTA 1줄 + 디스클레이머 1줄을 포함하라.
- 금지 표현(전문/최고/100%/반드시/무료상담/할인/승소율/결과보장/전관/비방/비교/식별 단서/ must_avoid)은 포함하지 말라.
- 외부 URL을 본문에 넣지 말라.

반환은 JSON 하나로 끝내라.
